Here Comes the Sun

This project involves using exoplanet data, putting the user at the center of the solar system.


Source Code

Link to the source code here.


Instructions


Download the project folder, and unzip. Go to unity and select "Open Project." All that needs to be done is to press play. To interact with the environment, use the pointers to press the interactable buttons appearing on the headset menu. Additionally, you can press the touchpad on objects in the linear space to teleport to those locations. To click a button on the menu, continue touching the touchpad and press the trigger. With this application, you can increase and decrease the sizes of the planets, increase and decrease the scale of the orbits, increase and decrease the speed of the planet revolution in the 3D space, and increase and decrease the scales present in the 2D view.
